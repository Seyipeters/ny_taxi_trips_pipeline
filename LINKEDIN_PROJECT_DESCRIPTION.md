# LinkedIn Project Description (Entry Level)

## NY Taxi ETL Pipeline | Python, Docker, BigQuery, Terraform

Built an entry-level cloud data engineering project that ingests NYC Taxi parquet data, transforms core trip fields, and prepares analytics-ready output for local storage and optional BigQuery loading.

### Highlights
- Built an ETL script in Python to extract and transform monthly taxi data.
- Standardized key columns (timestamps, numeric fields, quality filters) for cleaner analytics.
- Added optional BigQuery upload path for cloud warehouse integration.
- Added Docker support for reproducible execution.
- Added Terraform files to provision BigQuery dataset and table schemas.
- Added unit tests and a simple CI pipeline (lint, tests, and smoke check).

### Recruiter Keywords
AWS Cloud Data Engineering (transferable), ETL, Python, Data Pipeline, BigQuery, Terraform, Docker, Data Transformation, Analytics Engineering, CI/CD, Testing.

### Suggested LinkedIn One-Liner
Designed and shipped an end-to-end taxi ETL pipeline with Python, Docker, Terraform, and cloud warehouse loading, including tests and CI for reliable execution.
